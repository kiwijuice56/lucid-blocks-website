my game crashes or won't boot!

  • GPU-related fixes

    • if your game is crashing after updating it, delete the shader_cache and vulkan folders inside of the %APPDATA%/lucid blocks/ folder. also delete the device/device.tres file.
    • upgrade your graphics driver version. please do not use the built-in Windows driver tool. instead, look up the dedicated driver installer for your GPU (for example: Intel, AMD, NVIDIA) and use that tool instead.
    • if you've used programs like NVIDIA Control Panel or AMD Software, reset any 3D settings to their defaults. disable any advanced settings such as super-resolution and anti-lag.
    • switch to integrated graphics if your device supports it.
    • lower your graphics preset in-game.

  • other common fixes

    • if you have any mods installed, disable them by renaming the mods\ folder.
    • reinstall the game and verify Steam files.
    • disable your antivirus or add lucid blocks as an exception.
    • run the game from C:\Program Files (x86)\Steam\steamapps\common\lucid-blocks\lucid-blocks\lucid-blocks.exe in the file system rather than from the Steam client.
    • turn off Smart App Control on your device.

my display/input won't work!

  • can't click on Steam Deck

    switch to Proton 9.0! this is a known Steam bug.

  • can't fullscreen / taskbar won't go away

    press f11 to toggle fullscreen.

  • the window is too big for my screen

    press windows key + arrow key to forcefully resize the window. this should make the title bar visible for you to move and resize the window properly.

my qualia was corrupted!

  • locating backups

    don't panic, the game stores back-ups of the previous ten saves within the %APPDATA%/lucid blocks/backups/ folder. each subfolder corresponds to a snapshot of a qualia before you saved it. to find the matching backup, look through the backup_info.txt files inside each subfolder to check the name of the qualia (it'll look a little garbled, but you can find a piece of text like "s:qualia_name","s:[your qualia name]" as shown in the image below with a qualia titled home,home).

  • recovery

    before proceeding, copy all %APPDATA%/lucid blocks/ contents to a safe locations on your device in case you mess up the recovery process and need to start over.

    assuming you found the correct backup, open the backup_info.txt file to find the UUID of your save file (it should look like a string of random numbers and letters, such as a1a07956-bdd0-41ed-b3de-349e9a9598f).

    next, copy and paste the data.txt and register.txt files into %APPDATA%/lucid blocks/qualia/{the UUID you found}/ (or gift_qualia if you downloaded this qualia from the firmament). if you did this correctly, your save file will function properly the next time you start the game.
